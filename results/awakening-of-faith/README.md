## 교감 방법

[한문 텍스트 교감기(Classical Chinese Text Collator)](https://github.com/HMSeodam/Chinese-Text-Comparison)를 사용하여 LCS(Longest Common Subsequence) 알고리즘으로 글자 단위 대조를 수행하였습니다.

### 색상 가이드

- **검정색**: 두 판본에서 동일한 한자
- **빨간색**: 서로 다른 한자 (마우스 오버 시 상대 판본의 대응 글자 표시)
- **회색**: 문장부호·숫자·영어 등 비한자 (비교 제외)
