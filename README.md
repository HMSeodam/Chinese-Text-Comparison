한문 텍스트 교감기 (Classical Chinese Text Collator)

한문(漢文) 문헌의 여러 판본을 글자 단위로 대조하고 차이를 시각화하는 도구입니다.
교감학(校勘學) 연구 및 이본(異本) 대조 작업을 자동화하여 효율성을 극대화합니다.

✨ 주요 특징
가변 길이 차이 감지: LCS(최장 공통 부분 수열) 알고리즘을 통해 글자가 추가되거나 누락된 경우에도 일치 지점을 정확히 찾아냅니다.

한자 중심 비교: 문장부호, 숫자, 영어 등은 비교에서 제외(회색 표시)하고 오직 한자만 대조하여 통계의 정확도를 높였습니다.

광범위한 한자 지원: CJK 통합 한자(Extension A~G 포함) 및 한국, 중국(간/번체), 일본의 모든 한자를 인식합니다.

다양한 인코딩 대응: UTF-8은 물론 EUC-KR, Big5, Shift_JIS 등 다양한 텍스트 인코딩을 지원합니다.

보안 및 오프라인 동작: 서버 전송 없이 브라우저 내에서 모든 처리가 이루어져 개인정보와 자료를 안전하게 보호합니다.

🚀 실행 방법
1. 브라우저에서 실행 (권장)
본 저장소의 Download ZIP을 눌러 압축을 해제합니다.

교감 실행.html 파일을 더블클릭하면 즉시 실행됩니다. (Chrome, Edge 등 최신 브라우저 최적화)

2. 데스크톱 앱으로 실행 (Python)
Python 설치 시 **"Add Python to PATH"**를 체크해야 합니다.

install_and_run.bat을 실행하여 필요한 패키지를 설치하고 프로그램을 띄웁니다.

이후에는 run.bat으로 바로 실행 가능합니다.

💡 사용 방법
파일 업로드: 비교할 텍스트 파일들을 선택합니다. (3개 이상 가능, 인코딩 설정 지원)

조합 선택: 대조를 원하는 파일 쌍(A↔B, A↔C 등)을 체크합니다.

결과 확인: * 통계 정보: 한자 수, 동일/상이 글자 수, 차이 비율 제공.

4가지 보기 모드: 통합 보기, 파일 A/B 기준 보기, 나란히 보기(Side-by-Side).

색상 표기: 검정(동일), 빨간색(상이), 회색(비교 제외 문자).

저장: "HTML 저장" 버튼으로 결과 페이지를 소장할 수 있습니다.
