# 한문 텍스트 교감기 (Classical Chinese Text Collator)

한문(漢文) 문헌의 여러 판본을 글자 단위로 대조하고 차이를 시각화하는 도구입니다.

교감학(校勘學) 연구 및 이본(異本) 대조 작업을 자동화하여 연구의 효율성과 정확성을 극대화합니다.

> 🔧 **웹에서 바로 교감 실행**: [한문 텍스트 교감기 실행하기](https://hmseodam.github.io/Chinese-Text-Comparison/)
> 
> 📖 **교감 결과 바로 보기**: [한문 텍스트 교감 결과 페이지](https://hmseodam.github.io/Chinese-Text-Comparison/교감결과.html)

## 주요 특징

* **가변 길이 차이 감지**: LCS(Longest Common Subsequence) 알고리즘을 통해 글자가 추가되거나 누락된 경우에도 일치 지점을 정확히 찾아냅니다.
  
* **한자 중심 비교**: 문장부호, 숫자, 영어 등은 비교에서 제외(회색 표시)하고 오직 한자만 대조하여 통계의 정확도를 높였습니다.
  
* **광범위한 한자 지원**: CJK 통합 한자(Extension A~I 포함) 및 한국, 중국(간/번체), 일본의 모든 한자를 인식합니다.
  
* **다양한 인코딩 대응**: UTF-8은 물론 EUC-KR, Big5, Shift_JIS 등 다양한 국가의 텍스트 인코딩을 지원합니다.
  
* **보안 및 오프라인 동작**: 서버 전송 없이 브라우저 내에서 모든 처리가 이루어져 소중한 연구 자료를 안전하게 보호합니다.

* **PC·모바일 대응**: 교감 실행과 결과 열람 모두 PC와 모바일에서 최적화된 환경을 제공합니다.

## 사용 방법

#### **1단계: 파일 업로드**

* 비교할 텍스트 파일들을 선택합니다. (3개 이상 가능, 개별 인코딩 설정 지원)

#### **2단계: 조합 선택**

* 대조를 원하는 파일 쌍(A↔B, A↔C 등)을 체크하여 실행합니다.

#### **3단계: 결과 확인**

* **통계 정보**: 한자 수, 동일/상이 글자 수, 차이 비율을 한눈에 확인합니다.
  
* **4가지 보기 모드**: 통합 보기, 파일 A 기준, 파일 B 기준, 나란히 보기(Side-by-Side)
  
* **색상 가이드**:
  
    * **검정색**: 동일한 한자
      
    * **빨간색**: 서로 다른 한자 (PC: 마우스 hover 툴팁 / 모바일: 탭하면 하단 팝업)
      
    * **회색**: 문장부호 및 비한자(비교 제외)

#### **4단계: 결과 저장**

* `HTML 저장` 버튼을 눌러 결과 페이지를 파일로 소장할 수 있습니다.
